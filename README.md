## How to run

A Docker image is pushed to Docker Hub every time CI runs successfully.

> *Download the latest docker image*:

  docker pull jnahas/cosmos-ui

> *Open CLI and run*:

  docker run -it -p 7080:80 --rm --name telescope jnahas/cosmos-ui
